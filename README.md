# Task Management - Instrucciones de Despliegue

## Levantar el Frontend

Para iniciar la aplicación frontend, sigue estos pasos:

1. Asegúrate de haber levantado primero el backend. Consulta el README del backend para obtener instrucciones detalladas.

2. Abre una terminal en el directorio del frontend.

3. Ejecuta el siguiente comando para instalar las dependencias:

   ```bash
   npm i
   ```

4. Después de que se hayan instalado las dependencias, ejecuta el siguiente comando para iniciar el servidor de desarrollo:

   ```bash
   ng serve -o
   ```

5. Abre tu navegador web y visita la siguiente URL:

   [http://localhost:4200]

¡Listo! Ahora deberías poder acceder a la aplicación frontend de gestión de tareas. Si encuentras algún problema, asegúrate de haber seguido todas las instrucciones y revisa la consola de la terminal para obtener mensajes de error adicionales.
